git概述
        git是一个免费的、开源的分布式版本控制系统，可以快速高效地处理从小型到大型的各种项目
        git易于学习，占地面积小，性能极快。它具有廉价的本地库，方便的暂存区域和多个工作流分支

        vim hello git.txt 创建txt文件并编辑
	如果文件存在则直接进入 不存在则创建该文件并进入
        	输入 i 进入编辑状态
        	按下[esc]退出编辑
        	按下[:wq]表示保存退出
	不需要进入编辑状态按下yy复制 按下p粘贴

        vim hello.txt  创建文本

	添加到暂存区 git add hello.txt
	提交到本地库 git commit -m "hot-fix commit" hello.txt
	合并分支     git merge hot-fix
